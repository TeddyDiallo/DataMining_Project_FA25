To run this project

1) Get the diabetes_binary_50_50 dataset we've been working with.
2) Clone this repo and make sure that that data file is in the same directory as the DataProcessing.r file.
3) Run the setup.r file
4) Run the DataProcessing.r file to build the models off of the data.
5) Run the run_api.R file to start up the API server on port 9559.
6) Then navigate to the data-mining-project folder
7) Run npm i
8) Run npm start
9) Navigate to http://localhost:3000/
10) Try out the application
